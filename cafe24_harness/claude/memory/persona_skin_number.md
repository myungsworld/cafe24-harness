로컬 스킨 폴더 매핑 (이 몰):
- **원본 백업 스킨**: `__FILL__` (예: `skin20/` — 수정 금지, 참조용)
- **작업 스킨**: `__FILL__` (예: `skin22/` — 메인 작업 폴더, FTP/git 단위)

> 페르소나 플레이스홀더. 프로젝트 폴더 구조 확인 후 채운다.

**카페24 운영 측:**
- 로컬 작업 스킨 콘텐츠가 카페24 **"대표 디자인"** 으로 적용되어 운영된다.
- 어드민에선 *디자인 > 디자인 보관함*의 해당 스킨 항목으로 보이며, 카페24 내부 스킨 번호/이름은 로컬 폴더명과 별개일 수 있다.

**How to apply:**
- 로컬 작업/코드/가이드의 *파일 경로* 표현은 폴더명 그대로(`skin22/` 등).
- 클라이언트 안내(docs, Slack)에서 어드민 작업을 설명할 땐 "카페24 대표 디자인" 또는 "디자인 보관함의 해당 스킨"으로 표현 — 폴더명으로 쓰면 어드민에서 못 찾는다.

> ⚠️ 이미지 서빙 경로(`/web/upload/...`)는 스킨 번호와 별개 — [[persona_image_paths]] 참조.
